## Environment
- [Pipenv: A Guide to the New Python Packaging Tool](https://realpython.com/pipenv-guide/)

```shell
sudo apt install python3-pip
pip3 install pipenv

pipenv shell
pipenv install matplotlib==3.3.3
pipenv install pytest --dev

python3 -m venv venv
source venv/bin/activate.fish
pip install pipenv
pipenv shell --fancy


```
